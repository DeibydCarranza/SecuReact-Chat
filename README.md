# SecuReact-Chat
A secure message sending protocol using React and Socke.io


## Installation
```bash
# Install nvm-sh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash


# Raiz del proyecto
npm i cors express socket.io morgan
npm i nodemon -D


# Carpeta Frontend/
npm i vite
```
